Product Restriction on Users

This module will helps you to allow products to user or only access products related to allowed categories | Allow access of products/category to user



App Features
  • Allow only selected products to user
  • Allow only selected category of products to user
  • User can only access products allowed by Admin.
  • User can not see product while creating records like (Sale, Purchase, Inventory, Manufacturing, Invoice, POS, etc.)
  • Admin can access all products.
  • Also compatible with community and enterprise
